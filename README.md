ðŸ“‚ File: - SQL: [queries.sql](sql/queries.sql) â€¢ [insert_sales.sql](sql/insert_sales.sql)

Example queries:  

```sql
-- Total Orders
SELECT COUNT(order_id) AS total_orders FROM sales;

-- Unique Customers
SELECT COUNT(DISTINCT customer_id) AS total_customers FROM sales;

-- Top 5 Products by Revenue
SELECT product, SUM(revenue) AS total_revenue
FROM sales
GROUP BY product
ORDER BY total_revenue DESC
LIMIT 5;

-- Monthly Sales Trend
SELECT MONTH(order_date) AS month_no, SUM(revenue) AS monthly_sales
FROM sales
GROUP BY MONTH(order_date)
ORDER BY month_no;
```
